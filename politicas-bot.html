Política de Privacidade

Este bot respeita sua privacidade:

  • Não coletamos dados pessoais identificáveis.
  • Os dados coletados (como IDs de usuário/servidor) são usados apenas para funcionamento interno.
  • Não compartilhamos dados com terceiros.
  • Você pode solicitar a exclusão de dados entrando em contato com o desenvolvedor.
